#Koa Boilerplate

##Getting Started
```bash
$ git clone http://github.com/drGrove/koa-boilerplate projectName
$ vim package.json #Update with your projectName
$ cd public/swagger
$ bower install
$ cd ../../
$ npm install
$ node --harmony index.js
```
