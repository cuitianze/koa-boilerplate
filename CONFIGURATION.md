# Server Configuration Environment Variables

Below are a list of current environment variables that need to be set for the
application to run.
```bash
  NODE_ENV                # Node Environment ie. DEVELOPMENT,TESTING,PRODUCTION 
  APP_HOST                # Application Host ie. 0.0.0.0
  APP_PORT                # Application Port ie. 8888
  APP_NS                  # Application Namespace ie. /api/v1
  APP_DOMAIN              # Application Domain (for swagger reference)
  TOKEN_SECRET            # Application SALT secret
  TOKEN_LENGTH            # JWT Token Length
  GOOGLE_CLIENT_ID        # Google Auth Client ID
  GOOGLE_SECRET           # Google Auth Secret
  FACEBOOK_CLIENT_ID      # Facebook Auth Client ID
  FACEBOOK_SECRET         # Facebook Auth Secret
  GITHUB_CLIENT_ID        # Github OAuth client id
  GITHUB_CLIENT_SECRET    # Github OAuth secret
  LINKEDIN_CLIENT_ID      # Linkedin OAuth client id
  LINKEDIN_CLIENT_SECRET  # Linkedin OAuth secret
  LINKEDIN_PEOPLE_API_URL # Linkedin People API URL
  DATABASE_NAME           # Database Name
  DATABASE_USER           # Database User
  DATABASE_PASS           # Database Password
  DATABASE_DIALECT        # Database Dialect ie. mysql,mariadb
```
